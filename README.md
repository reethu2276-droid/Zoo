# Zoo Animals Project

A project to manage and organize information about zoo animals.

## Overview

This repository contains models, data, and utilities for managing a zoo's animal collection.

## Project Structure

- `animals/` - Animal models and classes
- `species/` - Species information and database
- `habitats/` - Habitat configurations and assignments
- `data/` - Sample data files
- `tests/` - Test suite

## Getting Started

See the individual directories for documentation on each component.

## Features

- Animal profile management
- Species classification
- Habitat assignment and tracking
- Animal health and feeding logs
